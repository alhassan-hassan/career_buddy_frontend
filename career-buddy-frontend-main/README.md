# Career Buddy User Frontend

User Frontend for the Career Buddy Application

## Requirements

- [nodejs](https://nodejs.org/en/download/)
- [vite](https://vitejs.dev/guide/)

## How to run

- `cd` into project directory via the terminal
- `npm install && npm run dev`
